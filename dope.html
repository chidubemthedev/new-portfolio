Dope Application

Website Web App Web Extension

Gamify your learning experience

The DOPE application is a browser extension that tracks logged-in users progress on specific learning sites and allocates points to them on completion. The users are able to log into their account on a DOPE platform to view their accumulated points and their current standing on available leaderboards.

It is a 2-part application that consists of both the broswer extension and the web app. The broswer extension keeps track of the user's progress and points. The web app contains the dashboard of a user with all informattion regarding them, including features like leaderboards, leagues, profile settings and many more.
